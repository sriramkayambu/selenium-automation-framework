# Selenium Automation Framework

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Apache POI
- Postman (API Testing)

## Framework Design
- Page Object Model (POM)
- Data Driven Testing using Excel
- TestNG for test execution and reporting

## Automated Modules
- Login
- Registration
- Product Search
- Add to Cart
- Checkout
- Logout

## How to Run Tests
1. Clone the repository
2. Import as Maven project in Eclipse
3. Run testng.xml or test classes
